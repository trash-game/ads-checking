# AdCheck – Kiểm tra nội dung quảng cáo

Plugin WordPress kiểm tra mức độ tuân thủ pháp lý của nội dung quảng cáo
(TPCN, Mỹ phẩm).

## Tính năng
- Phát hiện từ khóa quảng cáo vi phạm
- Chấm điểm 0–100
- Phân loại mức độ rủi ro
- Hỗ trợ nhiều lĩnh vực

## Cài đặt
1. Upload thư mục `adcheck` vào:
   `wp-content/plugins/`
2. Kích hoạt plugin trong WordPress
3. Dùng shortcode: `[adcheck_form]`
