# Công cụ kiểm tra nội dung quảng cáo

## Giới thiệu
Đề tài xây dựng công cụ kiểm tra nội dung quảng cáo trong lĩnh vực:
- Thực phẩm chức năng
- Mỹ phẩm

Hệ thống phát hiện từ khóa nhạy cảm, chấm điểm mức độ rủi ro và đưa ra cảnh báo theo quy định pháp luật Việt Nam.

## Chức năng chính
- Nhập nội dung quảng cáo
- Chọn lĩnh vực (TPCN / Mỹ phẩm)
- Chấm điểm 0–100
- Phân loại mức độ:
  - 80–100: Hợp lệ
  - 60–79: Cần chú ý
  - 40–59: Nguy cơ vi phạm
  - 0–39: Vi phạm cao
- Cảnh báo từ khóa vi phạm

## Công nghệ sử dụng
- PHP
- HTML/CSS
- WordPress Plugin
